# 🟢 What is Document Processing

* Refers to tools and techniques for converting unstructured data meant for human eye into structured data that can be searched, analyzed, and integrated into automated systems

<mark style="color:$danger;background-color:purple;">**Parsing ⇒ Markdown/HTML:**</mark>

* <mark style="color:$danger;background-color:purple;">**Convert pdf/png into machine readable text**</mark>
* <mark style="color:$danger;background-color:purple;">**OCR is a common approach for converting image to text**</mark>
* <mark style="color:$danger;background-color:purple;">**Preferred format is .md**</mark>
* Directly includes metadata such as coordinates of bounding boxes

<mark style="color:$danger;background-color:purple;">**Extraction:**</mark>

* <mark style="color:$danger;background-color:purple;">**Pull specific fields from parsed content ⇒ json prefered**</mark>
* JSON key-value pairs matching user-defined schemas
