# OCR

* It parses text, it doesnt understand the data
* 2 step process resulting in a digital output containing text

**Step 1: Image preprocessing**

* remove noise, correct tilted scans

**Step 2: Text recognition:**

* Advanced pattern matching compares image shapes against a library of known characters



**Limitations:**

* Loses document layout
* No understanding of data
* Poor performance with handwriting data
* Struggles with complex tables and charts
