# Getting better results

**Challenges:**

* Generic response: Prompt didn't include enough context
* The response is too long: Not provided length
* Didn't follow instructions: It understood what to be done, you need to provide info about how it needs to be done



**AI Fluency:**

* Ability to collaborate effectively with AI tools
* Four core competencies that, when combined, can help you make the most of your AI interactions:

1. **Delegation:** Deciding on what work should be done by humans, what work should be done by AI, and how to distribute tasks between them
2. **Description:** Includes clearly defining outputs, guiding AI processes, and specifying desired AI behaviors and interactions
3. **Discernment:** evaluating AI outputs, processes, behaviors and interactions
4. **Diligence:** Using AI responsibly and ethically. Includes making thoughtful choices about AI systems and interactions, maintaining transparency, and taking accountability for AI-assisted work.



**Evaluating Claude for your workflows:**

* Evals to test how well Claude performs on specific types of tasks that matter to you



**A simple eval approach:**

* Gather example of some work that you do regularly
* Create prompt to generate the same output
* Compare outputs
* Refine prompts based on the outputs
