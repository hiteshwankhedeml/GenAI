# Evaluate available NMT models

**Available Models:**

* **IndicTrans (2021)**
  * Open-source Indian-language NMT
  * Strongest for Indic languages
  * Trained on Samanantar
  * Supports 11+ Indic languages
  * Best choice for 2022
* **mBART50 (2020)**
  * Multilingual seq2seq model
  * Supports some Indic languages
  * General-purpose, weaker on Indic pairs
  * Good as a baseline
* **mT5 (2021)**
  * Text-to-text transformer
  * Handles many languages
  * Not specialized for translation
  * Performance below IndicTrans for Indic NMT
* **OPUS-MT (2020–2021)**
  * Open-source MarianMT models
  * English↔Indic models available
  * Data noisy, quality inconsistent
  * Useful for comparison
* **IIT Bombay EN–HI model**
  * Academic Hindi↔English baseline
  * Limited to a single language pair
  * Mainly for benchmarking
