# Agents

* GenAI Application ⇒ Give input to LLM and get output
* For queries like 'Get me todays news' then LLM does not have this info, to answer this it needs to be dependent on some tool
* The response from tool is known as context, and using this LLM will answer the query
* Here it is able to route autonomously for which query which tool needs to be called
* We used to do ReACT architecture for building agent
* Now it is simplified in LangChain 1.1
