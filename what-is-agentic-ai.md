# 🟢 What is Agentic AI

* <mark style="color:$danger;background-color:purple;">**AI systems that can act autonomously to achieve goals, not just respond to prompts**</mark>
* <mark style="color:$danger;background-color:purple;">**It can plan, decide, act, observe results, and iterate without constant human input**</mark>



**Non Agentic workflow:**

* Essay on topic X from start to finish in one go, without using backspace



**Agentic workflow:**

* <mark style="color:purple;background-color:purple;">**Process where an LLM based app executes multiple steps to complete a task**</mark>&#x20;
* Write an essay outline on topic X ⇒ LLM
* Do you need any web research ⇒ LLM + Websearch
* Write a first draft ⇒ LLM
* Consider what parts need revision or more research ⇒ LLM + Human request review
* Revise your draft ⇒ LLM



* It thinks/does research
* then again revises and so on
* So agentic workflow can take longer, but delivers much better results
